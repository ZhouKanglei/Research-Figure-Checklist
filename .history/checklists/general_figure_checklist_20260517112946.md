# 通用科研绘图 Checklist

## 1. 图的目的
- [ ] 这张图是否服务于一个明确结论？
- [ ] 读者是否能在几秒内看出主要信息？
- [ ] 这张图是否比表格更适合表达当前结果？

## 2. 数据表达
- [ ] 是否完整展示了关键对比？
- [ ] 是否避免选择性展示结果？
- [ ] 坐标轴范围是否合理？

## 3. 视觉设计
- [ ] 字体、字号、线宽是否统一？
- [ ] 图例是否清晰且不遮挡数据？
- [ ] 子图编号是否统一？

## 4. 配色
- [ ] 颜色是否有明确含义？
- [ ] baseline 和 ours 是否区分清楚？
- [ ] 是否避免过多颜色？

## 5. 投稿规范
- [ ] 是否保存为 PDF/SVG/EPS？
- [ ] 缩小到论文单栏/双栏后是否可读？
- [ ] caption 是否总结了核心发现？
